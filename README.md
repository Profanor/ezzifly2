## Overview
Ezzifly is a modern flight booking UI with authentication and profile management. Built with Next.js, TypeScript and Tailwind CSS.

## Features
- Home / Flight search (responsive)
- Register / Sign-in using backend auth (see Postman doc)
- Profile page (view & update personal details)
- Tailwind styling and responsive layouts

## Stack
- Next.js + TypeScript
- Tailwind CSS
- Axios for API calls
- react-hook-form + zod for form handling
